# SkyCast 

## Overview
SkyCast is a responsive weather application that provides real-time, location-based forecasts with a clean and minimal user interface.  
It utilizes the OpenWeather API to display accurate weather data including temperature, humidity, wind speed, and general weather conditions for any city worldwide.

The project focuses on simplicity, performance, and a seamless user experience using HTML, CSS, and JavaScript.

---

## Features
- Location-based weather forecasts using browser geolocation.
- City search functionality for worldwide weather lookup.
- Real-time updates of temperature, humidity, wind speed, and conditions.
- Dynamic UI that changes based on current weather conditions.
- Clean, minimal, and responsive design for all screen sizes.
- Fast and lightweight implementation using vanilla JavaScript.
